# Expense_Tracker
A simple Python project that helps users record daily expenses and view total spending. It allows adding and viewing expenses using basic Python concepts like variables, lists, and file handling. This project was built to practice core Python fundamentals and logical thinking.

## Functions of the Expense Tracker Project

1.Add a new expense with date, category, description, and amount.
2.Store expenses using a list of dictionaries.
3.View all recorded expenses in an organized format.
4.Calculate and display the total amount of expenses.
5.Menu-driven program for easy user interaction.
6.Input validation for basic user choices.
7.Exit the program safely when the user selects the exit option.
